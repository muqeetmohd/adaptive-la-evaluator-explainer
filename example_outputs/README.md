The same question — "What is an eigenvalue?" — answered by the system at each of the three tiers, showing how the adaptive explanation changes based on the student's diagnosed level.

| File | Tier | Audience |
|---|---|---|

| [eigenvalues_tier2.md](eigenvalues_tier2.md) | Tier 2 | Formal Beginner — definitions with concrete examples |

The tier is determined automatically from the student's answers to 3 diagnostic questions before they select a topic.
